# AMID

AMID is a Windows download manager for normal HTTP and HTTPS links.

## Install AMID

1. Open the latest release:

```text
https://github.com/dylanchromedome/amid/releases/latest
```

2. Download the newest zip, named like:

```text
AMID-portable-win-x64-v0.9.3.zip
```

3. Right-click the zip and choose `Extract All`.
4. Open the extracted folder.
5. Run:

```text
install.exe
```

6. Allow the Windows administrator prompt.
7. Open AMID from the Start menu by searching `AMID`.

## Install The Chrome Extension

1. Start AMID.
2. In AMID, open the `Options` tab.
3. Click `Copy Path` or `Open Folder` under `Chrome extension`.
4. Click `Copy Chrome URL`.
5. Open Chrome and paste this into the address bar:

```text
chrome://extensions
```

6. Enable `Developer mode`.
7. Click `Load unpacked`.
8. Select the AMID Chrome extension folder.

If AMID is installed normally, the folder is:

```text
C:\Program Files\AMID\chrome-extension
```

Chrome downloads are sent to AMID only while AMID is open. If AMID is closed, Chrome downloads normally.
